# Hospital
* Website for making an appointment with a doctor
## Technologies
* PHP
* Bootstrap
* phpMyAdmin
## Setup
* Open welcome.php in browser
## Functions
* You can create account as a patient or a doctor. You can make new appointment, remove them and add them to your calendar
